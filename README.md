### 使用方式

```shell
# 安装依赖 (如果没有全局安装过 pnpm, 需要先执行 npm i -g pnpm)
pnpm i
# 启动微信小程序开发模式
pnpm dev:mp-weixin
```

### vscode 推荐插件（建议使用 vscode 开发）

- **uni-helper**
- **Tailwind CSS IntelliSense**
- **Vue - Official**
- **ESLint**

### 升级

```shell
# 文档 https://zh.uniapp.dcloud.io/quickstart-cli.html#cliversion
npx @dcloudio/uvm@latest
```
